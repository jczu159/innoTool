# 遊戲對照工具（MegaXcess Mapping Tool）

## 功能簡介

此工具用於將第三方遊戲商的 PDF 遊戲清單，自動解析並寫入資料庫的 `tiger_thirdparty.megaxcess_game_type_mapping` 表。

只需指定 **Provider** 並上傳對應的 **PDF 遊戲清單**，工具會自動：
1. 解析 PDF 取得遊戲代碼、名稱、類型
2. 與 DB 中 `slot_game_setting_i18n` 的遊戲代碼進行比對
3. 預覽比對結果（新增 / 更新 / 略過）
4. 一鍵寫入 DB 或匯出 SQL 檔案

---

## 啟動方式

```
D:\tiger-release-helper\dist\遊戲對照工具\遊戲對照工具.exe
```

---

## 使用流程

1. **填寫 DB 連線資訊**（預設已填入開發環境）
   - Host / Port / User / Password
   - 按 **Test DB** 確認連線是否正常

2. **選擇 Provider**（下拉選單）

3. **選擇 PDF 檔案**（按 Browse）

4. 按 **Parse PDF** 開始解析

5. 預覽結果表格：
   - 🟢 **NEW** — DB 無此遊戲，將新增
   - 🟡 **UPDATE** — DB 已有，將更新
   - ⚪ **SKIP** — 無變動

6. 確認後：
   - **Apply to DB** — 直接寫入資料庫
   - **Export SQL** — 匯出 `.sql` 檔案供人工執行

## 對照規則

| PDF 分類            | game_offering | game_type        |
|--------------------|---------------|------------------|
| eCASINO GAMES      | EGAMES        | SLOT / TABLE 等  |
| eBINGO GAMES       | EBINGO        | BINGO            |
| eSPORTS GAMES      | SPORTSBOOK    | SPORT            |
| 其餘               | EGAMES        | SLOT（預設）     |

- **CODE_EXACT**：DB game_code 與 PDF game_id 完全一致 → 使用 PDF 資料
- **NOT_IN_PDF**：DB 有但 PDF 無 → 保留 EGAMES/SLOT
- **PDF_NOT_DB**：PDF 有但 DB 無 → 略過（僅供參考）
